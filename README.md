# 🌐 CV Web - Proyecto de Curriculum Vitae Online

Este proyecto consiste en la creación de un Curriculum Vitae (CV) en formato web, desarrollado con HTML, CSS y JavaScript, con el objetivo de ser desplegado en GitHub Pages.

Además de mostrar el resultado final, este repositorio documenta el **proceso de creación asistido por IA**, optimizando resultados mediante prompts bien diseñados.

---

## 🚀 Objetivo

Crear una página web simple, moderna y responsive que represente un CV profesional, utilizando buenas prácticas de desarrollo frontend y asistencia de herramientas de inteligencia artificial.

---

## 🧠 Proceso de creación

### 1. Generación del prompt

El primer paso fue diseñar un prompt detallado para utilizar en un modelo de IA (Claude), con el objetivo de obtener una base sólida del proyecto.

Se definieron requisitos como:
- Estructura HTML5 semántica
- Estilos modernos con CSS3
- Responsive design (mobile-first)
- Uso de Google Fonts
- Implementación de modo claro/oscuro
- Interactividad con JavaScript
- Organización de archivos clara y mínima

Esto permitió **maximizar la calidad del código generado desde el inicio**.

---

### 2. Generación del proyecto base con IA

Se utilizó el prompt en Claude para generar:
- `index.html`
- `styles.css`
- `script.js`

El resultado fue una primera versión funcional del CV web con:
- Secciones estructuradas
- Estilos básicos aplicados
- Funcionalidad inicial de navegación

---

### 3. Ajustes y mejoras manuales

Una vez generado el código base, se realizaron mejoras para personalizar y profesionalizar el proyecto:

#### ✏️ Contenido
- Se agregaron descripciones más realistas en la sección "Sobre mí"
- Se mejoró la sección de experiencia/formación
- Se ajustaron las habilidades con mejor presentación visual

#### 🖼️ Recursos visuales
- Se incorporó una foto de perfil
- Se ajustaron tamaños y estilos de imagen

#### 🔗 Enlaces
- Se añadieron links a redes sociales (LinkedIn, GitHub, etc.)
- Se mejoró la navegación interna

#### 🎨 Estilos
- Ajustes en colores y tipografía
- Mejoras en spacing y layout
- Refinamiento del modo oscuro

#### ⚙️ JavaScript
- Mejora del scroll suave
- Ajuste del comportamiento del botón "volver arriba"
- Optimización del toggle de tema claro/oscuro
